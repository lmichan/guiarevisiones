LAs revisiones de literatura constituyen la forma mas completa y sofisticada de reunir el conocimiento científico, consisten en investigaciones que utilizan publicaciones científicas, generalmente artículos para generar nuevo conocimiento a partir del análisis, la síntesis, la integración y la estadísitca existen varios tipos de revisiones que se pueden agrupar con base en varios criterios.

En esta guía queremos exponer de manera detallada los elementos básicos y avanzados ue todo usuario debe conocer para utilizar las revisiones científicas con cluides, para ello expondremos las caracteríticas de este tipo de literatura, las tecnologías informáticas para procesarlas, así como las aplicaciones que tienen acompañadas de ejemplos.

Clasificaciones de las revisiones
1. Narrativas y Sistemáticas
2. Estáticas y Actualizadas
3. Cualitativas y cuantitativas
4. Sintesis e Impacto

Esta guía está diseñada para exponer cada una de las revisiones con base en cinco elementos:
1. Tipos
2. Descripción
3. Caracteríticas
4. Aplicaciones
5. Ejemplos
